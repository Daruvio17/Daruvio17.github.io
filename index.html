Modelo de precios de toyota - Análisis Profundo

Este proyecto analiza el impacto de variables como kilometraje, año y tipo de combustible en el valor de reventa de autos Toyota.

Especificación del Modelo:

$$ \ln(Price)_i = \beta_0 + \beta_1 Age_i + \beta_2 KM_i + \beta_3 HP_i + \epsilon_i $$

Código Destacado

# Estimación del modelo por MCO
modelo_toyota <- lm(log(Price) ~ Age_08_04 + KM + HP + Weight, data = toyota_data)
summary(modelo_toyota)
