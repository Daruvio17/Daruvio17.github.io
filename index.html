Modelo de precios de Toyota - Análisis Profundo

Este proyecto analiza el impacto de variables como kilometraje, año y tipo de combustible en el valor de reventa de autos Toyota.

Exploración de Datos (Head)

Visualización de las primeras filas del dataset para entender la estructura de las variables antes del modelado:

Figura 1: Muestra de datos importados desde Excel a R.

Especificación del Modelo Completo:

$$ \text{Precio}_i = \beta_0 + \beta_1 \text{Edad\_Meses}_i + \beta_2 \text{KM}_i + \beta_3 \text{HP}_i + \dots + \beta_{32} \text{Asistente\_Parqueo}_i + \epsilon_i $$

Nota: El modelo incluye 32 variables explicativas, abarcando características técnicas (HP, Cilindraje), seguridad (ABS, Airbags) y confort. Contamos con un total de 1436 observaciones.

Para esta primera estimación, presentamos el modelo sin realizar aún la validación exhaustiva de supuestos. Es decir, trabajamos bajo el marco de las propiedades de Gauss-Markov, asumiendo ausencia de multicolinealidad perfecta y errores homoscedásticos:

$$ E(\epsilon_i | X) = 0 \quad \text{y} \quad \text{Var}(\epsilon_i | X) = \sigma^2 $$

Bajo estos supuestos, aseguramos que nuestros estimadores MCO sean los MELI (Mejor Estimador Lineal Insesgado).

Resultados de la Estimación (Summary)

A continuación se detallan los coeficientes estimados, errores estándar y niveles de significancia:

Figura 2: Salida oficial del comando summary(modelo) en R.

Interpretación Económica de algunos de los Coeficientes

Variables Continuas

  • Edad (Meses): Por cada mes adicional de antigüedad, el precio disminuye en promedio $0.118 dólares, manteniendo el resto constante.
  • KM: Por cada kilómetro adicional recorrido, el precio cae aproximadamente $0.00002 dólares.
  • HP (Caballos): Un incremento de un caballo de fuerza se traduce en un aumento promedio de $0.0196 dólares en el valor.
  • Peso: Por cada kilogramo adicional de peso (robustez), el precio tiende a subir $0.0107 dólares.
  • Impuesto Anual: Un aumento de $1 en el impuesto se asocia a una reducción de $0.014 dólares en el precio de reventa.

Variables Categóricas (Dummies)

  • Transmisión Automática: Un Toyota Corolla automático es, en promedio, $352.63 más costoso que uno de transmisión manual, ceteris paribus.
  • Tipo de Combustible (Petrol): Los autos a gasolina (Petrol) tienen un precio promedio superior en $2,050.61 respecto a la categoría de referencia, siendo este un diferencial crítico en el modelo.
  • Aire Acondicionado Automático: Esta característica de lujo aporta un incremento sustancial de $2,397.95 al valor del vehículo frente a los que no cuentan con este sistema.

Nota de Inferencia: Todas las interpretaciones se realizan bajo el supuesto Ceteris Paribus (manteniendo todo lo demás constante).

Pruebas de Hipótesis Individuales ($\alpha = 5\%$)

Para cada coeficiente, evaluamos si la variable tiene un impacto real en el precio mediante el siguiente contraste:

$$ H_0: \beta_j = 0 \quad \text{vs} \quad H_1: \beta_j \neq 0 $$
Variable P-Valor (Pr > |t|) Regla de Decisión Conclusión
Edad Meses 0.00000 p < 0.05$ Rechazo H_0 (Significativa)
KM 0.00000 p < 0.05$ Rechazo H_0 (Significativa)
Met_Color 0.31597 p > 0.05$ No Rechazo H_0 (No Sig.)
Automático 0.00868 p < 0.05$ Rechazo H_0 (Significativa)
Airbag_1 0.69563 p > 0.05$ No Rechazo H_0 (No Sig.)

Interpretación: Al usar un α = 0.05$, cualquier p-valor menor a este umbral nos permite afirmar con un 95% de confianza que la variable efectivamente afecta el precio del Toyota Corolla.

Código Implementado en R

library(readxl)
datos <- read_excel(file.choose())
datos <- data.frame(datos)
head(datos)
datos$precio <- datos$precio/1000
attach(datos)
modelo_inicial <- lm(precio ~., data = datos)
round(coef(summary(modelo_inicial)),5)
formula(modelo_inicial)
